# Vexchange API
  
## Description
The readonly API for Vexchange, the Leading DEX on VeChain.

## Installation
The use of NVM (node version manager) is recommended.

```bash
$ nvm use
$ npm install
```

## Running the app

```bash
# replicate .env
$ cp .env.example .env

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
